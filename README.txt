# Data Mining (COL761)

Part A) Apriori implementation: The file apriori.cpp contains an implementation of the apriori algorithm wherein we initially generate a set of frequent items and then until the candidate set is empty we keep generating frequent itemsets of size 1 greater as we do in apriori and filter to obtain frequent itemsets.
